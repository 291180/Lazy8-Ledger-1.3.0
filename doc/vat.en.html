VAT report
This report is basically only to produce figures for the Swedish Momsdeklaration and also only if you use the EU BAS 2000 account system. However, you can change this report to suit your needs. See Advanced->Report->Special report. The name of this report when you use Special report is 'VAT.en'

Company

This report will only do one company at a time.  So, you must choose the company to printout on here.  If you have only one company and have choosen to not be able to change the company (see Options->SETUP) then this will be disabled and you dont need to do anything here.

Start date

This report will print all transaction in the accounts between the given start and stop date where the start and stop date will be applied to the Invoice date field you entered in DataEntry->Transaction.  There must exist records bettween these two dates, otherwise there will be no report.

Stop date

This report will print all transaction in the accounts between the given start and stop date where the start and stop date will be applied to the Invoice date field you entered in DataEntry->Transaction.  There must exist records bettween these two dates, otherwise there will be no report.

Report title

This is the title of the report that will only appear at the top of the report.  This could be blank if you wish

Get report Button

Press this button when all the previous fields are entered.  The report will appear in a seperate word processing window

Get report into clipboard Button

Press this button when all the previous fields are entered.  The report will be directly put onto the clipboard. You must then paste the report into some other wordprocessor or spreadsheet. If you paste into a wordprocessor, then you will attain the best results if after you have pasted into the wordprocessor, that you select the entire report and then select -convert to table- in the wordprocessing program


This report is especially wide so I would suggest you make the word processing window as wide as possible, otherwise word wrapping will ruin how the report looks.
